= dagger-cli

Packages the Dagger CLI into an OCI Image for use within your CI. Built upon the https://github.com/wolfi-dev/os[Wolfi OS], both `amd64` and `arm64` architectures are included.

== Installed Packages

- bash
- ca-certificates-bundle
- dagger
- docker-cli
- gh
- git
- glab
- jq
- wolfi-base
